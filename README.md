# game-bot
 Игровой бот с угадыванием случайного числа (Game bot with guessing random number).
В файле game bot.html  размещен один элемент конпка (<input type="button") с прописанным событием в виде onclick="" В котором содержится функция tryToGuess().
1. В файле gb.js прописана механика работы.
2. Услвоия игры: Загадывается случайное число от 1 до 100. И дается 10 попыток, что бы угадать. Игра идет до нажатия ОТМЕНА/ESC. 
3. Условия генерации проверяются в строке 4 (если выпадет 0 то присваивается значение 1 в строке 5).
4. Условия игры (попытки и диапазаон чисел) и диапазон чисел меняется в 3 строке и 12 строке файла gb.js.
